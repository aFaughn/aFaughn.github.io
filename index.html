Humanitsy

Humanitsy is a (loose) clone of Etsy, Users can create their own products with a custom image and statistics, as well leave reviews on other products! Humanitsy was built using Python, Flask, JavaScript, React.js, and Redux.

Live SiteGitHub

TableOpen

TableOpen is a clone of OpenTable (we really brainstormed the title). Users are able to create restaurants, as well as leave reviews, add restaurants to their favorites, and make reservations! TableOpen was built using Python, Flask, JavaScript, React.js, and Redux.

Live SiteGitHub

Inscryptr

Inscryptr is based off of Flickr, the photo sharing application. In Inscryptr, you can create your own cards with custom card art and stats, view those stats, and share them with other users. You can also leave reviews on other people's cards. Inscryptr was built using postgreSQL, Express.js, React.js, and Redux

Live SiteGitHub
